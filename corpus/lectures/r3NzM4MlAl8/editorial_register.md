# Editorial Register: Layer 2 → Layer 3

*Lecture 5 (3.371 Welding Metallurgy, Summer 2015). This register records editorial judgments made in lifting the cleanup edit to the readable edition.*

## Conventions applied

Standard conventions from the catalog: student attribution, italicized stage directions, numerals per spec, smoothed self-corrections, filler removed, "okay" preserved where it closes thoughts, soft reordering within sections only.

## Sectioning rationale

Ten sections were used for this ~58-minute lecture. The natural breaks are:
- s1 → s2: pivot from Five Whys framing to soda-can demonstration ("So first we're going to start with an experiment today")
- s2 → s3: pivot from hydrogen physics back to the hardness/hardenability lecture proper
- s3 → s4: Tom's explicit aside ("breaks the train of thought on steel… it's a good story")
- s4 → s5: explicit return ("Okay, so hardenability, on the other hand…")
- s5 → s6: closure on tempering, return to weld defect taxonomy
- s6 → s7: pivot from defect taxonomy to Charpy toughness and the Liberty Ship report
- s7 → s8: from Charpy machine mechanics to the Coast Guard / Quincy LNG story
- s8 → s9: pivot to steel composition fingerprinting (different defense scenario)
- s9 → s10: from fingerprinting cases to the untempered-plate / fastener-paperwork case

## Decisions that are *not* permitted at layer 3

- Adding new sentences in Tom's voice (Feynman/Leighton/Sands could because Feynman was alive to ratify; we cannot).
- Reordering content across section boundaries.
- Merging cases Tom kept separate, or splitting cases he handled as one.
- Removing factual errors Tom made on tape. (Captioner errors are corrected silently; Tom's slips are bracketed; Tom's genuine misstatements are preserved and footnoted in layer 3 only when they would actively mislead a reader.)
