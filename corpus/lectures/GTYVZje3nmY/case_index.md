# Case Index — Lecture 5 (3.371 Welding Metallurgy, Spring 2014)

This session is a pure technical exposition on stainless steel weldability — closing notes on austenitic sensitization, then ferritic alloys (composition, applications, welding, embrittlement). No industrial, forensic, or historical cases are invoked. No case-cluster entries to record.

## Figures and named references

### Suutala diagram (and modified Suutala diagram)
- **Anchors:** `§2.p2`, `§2.p3`
- **Type:** technical reference (named diagram), not a case
- **Frame in this section:** The cracking-susceptibility diagram for austenitic stainless welds, indexed on chromium and nickel equivalents. Tapio Suutala (Finland, 1980s). The "modified" version accounts for the shift in solidification mode under high-energy-density welding (laser, electron beam) — threshold shifts from 1.5 to roughly 1.5–1.6 on the Cr/Ni equivalent axis.

### Schaeffler diagram
- **Anchor:** `§4.p1`
- **Type:** technical reference (named diagram), not a case
- **Frame in this section:** Mentioned in passing as one of the standard predictive diagrams for residual phases in welded stainless steels. The speaker had introduced it earlier in the lecture series.

## Generic industrial applications referenced (not cases)

- **Automotive exhaust systems / 409 mufflers** (`§3.p7`, `§3.p8`) — used to illustrate the typical application of low-chromium ferritic stainless steel, including the trade-off that 409 will eventually rust but outlasts plain carbon steel. Not a forensic case, not tied to a specific manufacturer or failure.
- **Automotive trim** (`§3.p7`) — generic application reference for ferritic stainless steel.
- **E-BRITE proprietary high-chromium alloy** (`§3.p6`) — product reference, mentioned as exemplar of the high-chromium ferritic class.

## Alloy designations mentioned

321, 347 (stabilized austenitic, §1.p2); 430 (medium-chromium ferritic workhorse, §3.p5); 409 (low-chromium ferritic, automotive muffler, §3.p8); E-BRITE (high-chromium proprietary ferritic, §3.p6).
